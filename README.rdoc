# README

## Dependencies

In order to run the application locally the following software is required:

- Ruby 2.2.2
- Bundler 1.10
- PostgreSQL 9.4.4

## Installation

1. Install dependencies
- Go to the project directory
- Run `bundler install`
- Run `rake db:setup`
- Run `rails server`
- Open `http://localhost:3000` in the browser
